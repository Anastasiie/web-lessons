Розмітка заголовків

Заголовки впорядковують текст, формуючи його візуальну структуру. При використанні заголовків необхідно враховувати їх ієрархію. В заголовки не можна вкладати інші елементи.

<h1> - заголовок найвищого рівня, на сторінці рекомендовано використовувати не більше одного разу

<h2>, <h3>, <h4>, <h5>, <h6> - заголовки нижчих рівнів, розташовані за спаданням їх важливості

Синтаксис:

Теги для форматування тексту

<h1> - являє собою найбільш важливий заголовок першого рівня, покликаний акцентувати увагу пошукових систем на певних ділянках тексту.

Синтаксис:

<h1> Тег визначає найбільш важливі заголовки </h1>

Результат:

Тег визначає найбільш важливі заголовки

<h2> - являє собою другий по-важливості заголовок (другого рівня). Текст усередині тега відображається великим шрифтом з товстими рисами.

Синтаксис:

<h2> Являє собою заголовок другого рівня. </h2>

Результат:

Являє собою заголовок другого рівня.

<h3> - являє собою третій по-важливості заголовок (третього рівня).

Синтаксис:

<h3> Встановлює заголовок третього рівня. </h3>

Результат:

Являє собою заголовок третього рівня.

<h4> - являє собою четвертий по-важливості заголовок (четвертого рівня).

Синтаксис:

<h4> Встановлює заголовок четвертого рівня. </h4>

Результат:

Являє собою заголовок четвертого рівня.

<h5> - являє собою п'ятий по-важливості заголовок (п'ятого рівня).

Синтаксис:

<h5> Встановлює заголовок п'ятого рівня. </h5>

Результат:

Являє собою заголовок п'ятого рівня.

<h6> - являє собою шостий по-важливості заголовок (шостого рівня).

Синтаксис:

<h6> Встановлює заголовок шостого рівня. </h6>

Результат:

Являє собою заголовок шостого рівня.

<strong> - виділяє важливий текст, відображається жирним текстом.

Синтаксис:

Тег визначає <strong>важливість</strong> визначеного тексту.

Результат:

Тег визначає важливість визначеного тексту.

<em> - призначений для тексту, що має особливе значення або на який слід наголосити, відображається курсивом.

Синтаксис:

Тег виділяє <em>важливі</em> фрагменти тексту, показуючи їх курсивом.

Результат:

Тег виділяє важливі фрагменти тексту, показуючи їх курсивом.

<small> - призначений для зменшення розміру шрифту на одиницю порівняно із звичайним текстом.

Синтаксис:

Тег <small>зменшує розмір</small> фрагменту текста.

Результат:

Тег зменшує розмір фрагменту текста.

<ins> - призначений для виділення тексту підкресленням. Застосовується для виділення змін, внесених у документ.

Синтаксис:

Тег виділяє текст <ins>підкресленням</ins>.

Результат:

Тег виділяє текст підкресленням.

<del> - призначений для виділення тексту, видаленого в новій версії документа. Позначають текст як перекреслений.

Синтаксис:

Тег позначає текст як видалений,<del>перекреслюючи</del> його.

Результат:

Тег позначає текст як видалений, перекреслюючим його.

<sub> - призначений для підрядкового написання символів, наприклад індексу елемента в хімічних формулах.

Синтаксис:

Тег позначає підрядкове написання символів, наприклад спирту - C<sub>2</sub>H<sub>5</sub>OH.

Результат:

Тег позначає підрядкове написання символів, наприклад спирт - C2H5OH.

<sub> - призначений для відображання шрифту у вигляді верхнього індексу. Шрифт при цьому відображається вище базової лінії тексту та зменшеного розміру.

Синтаксис:

Тег позначає надрядкове написання символів, x<sup>2</sup> + y = 1.

Результат:

Тег позначає надрядкове написання символів, x2 + y = 1.

Розмітка коду

<code> - призначений для відображення одного або декількох рядків тексту, який є програмним кодом.

Синтаксис:

Тег позначає відображення програмного коду, <code>if (i % 2 == 0) return y</code>.

Результат:

Тег позначає відображення програмного коду, if (i % 2 == 0) return y.

<kbd> - призначений для позначення тексту, який набирається на клавіатурі або назви клавіш.

Синтаксис:

Тег виділяє назви клавіш <kbd>Ctrl</kbd> + <kbd>Delete</kbd>.

Результат:

Тег виділяє назви клавіш Ctrl + Delete.

<samp> - призначений для відображення тексту, який є результатом виведення комп'ютерної програми чи скрипту.

Синтаксис:

<samp>Системне повідомлення</samp>.

Результат:

Системне повідомлення.

<pre> - визначає блок попередньо форматованого тексту. Такий текст зазвичай відображається моноширинним шрифтом і з усіма пробілами між словами.

Синтаксис:

<pre> - Системне повідомлення - </pre>.

Результат: 

 - Системне повідомлення -

<dfn> - застосовується для виділення нових термінів, та при їх першій появі у тексті. Відображаються за допомогою курсиву.

Синтаксис:

<dfn>Системне повідомлення </dfn>- це ...

Результат:

Системне повідомлення - це ...

Цитати, означення, аббревіатури

<abbr> - показує, що послідовність символів є абревіатурою.

Синтаксис:

<abbr>CGI*</abbr> позначається протокол, за допомогою якого будь-які зовнішні програми взаимодіютб з веб-сервером.

Результат:

CGI* позначається протокол, за допомогою якого будь-які зовнішні програми взаїмодіють з веб-сервером. CGI*

<q> використовується для виділення у тексті цитат. Вміст автоматично відображається у лапках.

Синтаксис:

Амелия Эрхарт, <q>Сложнее всего начать действовать, все остальное зависит только от упорства.</q>- це ...

Результат:

Амелия Эрхарт, Сложнее всего начать действовать, все остальное зависит только от упорства.

<cite> - представляє цитування назви творчої роботи (книги, статті, поеми, сценарію, фільми, пісні, опери, ігри та ін.).

Синтаксис:

Культова рольова гра <cite>Star Wars: Knights of the Old Republic</cite>.

Результат:

Культова рольова гра Star Wars: Knights of the Old Republic.

<blockquote> - призначений для виділення довгих цитат усередині документа.

Синтаксис:

Ричард Бах говорил : <blockquote>Каждый человек, появляющийся в твоей жизни, все события, которые с тобой происходят, — все это случается с тобой потому, что это ты притянул их сюда. И то, что ты сделаешь со всем этим дальше, ты выбираешь сам.</blockquote>

Результат:

Ричард Бах говорил :

Каждый человек, появляющийся в твоей жизни, все события, которые с тобой происходят, — все это случается с тобой потому, что это ты притянул их сюда. И то, что ты сделаешь со всем этим дальше, ты выбираешь сам.

Розриви

<br> - встановлює переведення рядка в тому місці, де він знаходиться.

Синтаксис:

Как бессонница в час ночной <br>Меняет, нелюдимая, облик твой, <br>Чьих невольница ты идей? <br>Зачем тебе охотиться на людей?

Результат:

Как бессонница в час ночной
Меняет, нелюдимая, облик твой,
Чьих невольница ты идей?
Зачем тебе охотиться на людей?

<hr> - малює горизонтальну лінію

Синтаксис:

<hr>

Результат:

Списки

<ul> - встановлює маркований список. Кожен пункт списку повинен починатися з елемента <li>

Синтаксис:

<ul> <li>Марія</li> </ul> <ul> <li>Амелія</li> </ul> <ul> <li>Камілла</li> </ul>

Результат:

  • Марія
  • Амелія
  • Камілла
<ol> - встановлюється з нумерованого списку. Кожен пункт списку повинен починатися з елемента <li>

Синтаксис:

<ol> <li>Марія</li> </ol> <ol> <li>Амелія</li> </ol> <ol> <li>Камілла</li> </ol>

Результат:

  1. Марія
  2. Амелія
  3. Камілла
<li> - визначає окремий пункт списку.

Синтаксис:

<li>Колір</li> ><li>Матеріал</li>

Результат:

  • Колір
  • Матеріал

Синтаксис:

<a href ="<адресa>">Назва></a>

Результат:

Назва

Зображення

<img> - призначений для відображення на веб-сторінці зображень у графічному форматі GIF, JPEG, SVG або PNG.

Синтаксис:

<img src="Space.png" alt="<текст>"> </img>

Результат:

&nbsp; - нерозривний пропуск.

Синтаксис:

Текст має три&nbsp;&nbsp;&nbsp;- пропуски

Результат:

Текст має три   - пропуски

&copy; - знак copyright.

Синтаксис:

Текст має Genius focus&copy;

Результат:

Текст має Genius focus©

&trade; - Знак торговой марки.

Синтаксис:

Спонсовано Genius focus&trade;

Результат:

Спонсовано Genius focus™

&amp; - Знак Амперсанд.

Синтаксис:

Спонсовано Genius&amp; focus

Результат:

Спонсовано Genius & focus

&lt; - Знак "менше ніж" (початок тега) .

Синтаксис:

1&lt;2

Результат:

1<2

&gt; - Знак "більше ніж" (кінець тега) .

Синтаксис:

2&gt;1

Результат:

2>1
