Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифту задається за допомогою значення та одиниць виміру (em, px, pt, %)

Синтаксис:

Css

p{
        font-size: 20px;
    }

Html

<p> Hello, world! </p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.

Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно батківського стилю в меншу сторону
  • bolder - змінює насиченість відносно батківського стилю в більшу сторону
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис:

Css

.bold{
        font-weight: bold;
    }
    .thin{
        font-weight: 100;
    }
    .max{
        font-weight: 900;
    }

Html

     <p>
            <span class="bold">Напівжирний текст, </span>
            <span class="thin">Найтонший текст, </span>
            <span class="max">Найтовщий текст.</span>
    </p>

Результат:

Напівжирний текст, Найтонший текст, Найтовщий текст.

Властивість font-style

Властивість font-style визначає яким чином шрифт повинен бути стилізований.

Може приймати наступні значення:

  • normal - нормальна версія звичайного шрифту
  • italic - курсивна версія шрифту
  • oblique - похила версія регулярного шрифту

Синтаксис:

Css

     .normal {
            font-style: normal;
    }
    .italic {
            font-style: italic;
    }
    .oblique {
            font-style: oblique;
    }

Html

        <p><span class="normal">Цей параграф нормальний. </span></p>
        <p><span class="italic">Цей параграф виділено курсивом.</span></p>
        <p><span class="oblique">Цей параграф є косим.</span></p>

Результат:

Цей параграф нормальний.

Цей параграф виділено курсивом.

Цей параграф є косим.

Властивість font-family

Властивість font-family визначає пріоритетний список із назв сімейства шрифтів та загальне ім'я шрифту для вибраного елемента.

Може приймати наступні значення:

  • serif - Гліфи мають завершальні штрихи, що розширюються або звужуються кінці, або мають фактично зазубрені закінчення.
  • sansserif - Гліфи мають гладкі закінчення.
  • monospace - Усі гліфи мають однакову фіксовану ширину.
  • cursive - Гліфи в курсивних шрифтах зазвичай мають або сполучні штрихи, або інші рукописні характеристики.
  • fantasy - Фентезійні шрифти - це насамперед декоративні шрифти, які містять грайливу виставу персонажів.
  • emoji - Шрифти спеціально призначені для відображення емодзі.
  • math - Це сімейство призначене для особливих стилістичних завдань уявлення математики: верхній індекс та нижній індекс, дужки, ...
  • fangsong - Особливий стиль китайських ієрогліфів, цей стиль часто використовується для урядових документів.

Синтаксис:

Css

     .serif {
        font-family: Times, Times New Roman, Georgia, serif;
    }
    .sansserif {
        font-family: Verdana, Arial, Helvetica, sans-serif;
    }
    .monospace {
        font-family: Lucida Console, Courier, monospace;
    }
    .cursive {
        font-family: cursive;
    }
    .fantasy {
        font-family: fantasy;
    }
    .emoji {
        font-family: emoji;
    }
    .math {
        font-family: math;
    } 
    .fangsong {
        font-family: fangsong;
    }

Html

        <p><span class="serif">This is an example of a serif font.</span></p>
        <p><span class="sansserif">This is an example of a sans-serif font.</span></p>
        <p><span class="monospace">This is an example of a monospace font.</span></p>
        <p><span class="cursive">This is an example of a cursive font.</span></p>
        <p><span class="fantasy">This is an example of a fantasy font.</span></p>

Результат:

This is an example of a serif font.

This is an example of a sans-serif font.

This is an example of a monospace font.

This is an example of a cursive font.

This is an example of a fantasy font.

Тип color

Тип color визначає колір у спектрі кольорів sRGB.

Синтаксис:

Css

     .color_p{
        color: #cf920c
    }

Html

<p><span class="color_p">This is an example of a color.</span></p>

Результат:

This is an example of a color.

Властивість text-transform

Властивість text-transform керує перетворенням тексту елемента у великі символи. Коли значення відрізняється від none, регістр вихідного тексту буде змінено.

Може приймати наступні значення:

  • capitalize- Перший символ кожного слова в реченні буде великим. Інші символи свого вигляду не змінюють.
  • lowercase- Усі символи тексту стають малими (нижній регістр).
  • uppercase- Усі символи тексту стають великими (верхній регістр).
  • none- Не змінює регістр символів.
  • inherit- Наслідує значення батька.

Синтаксис:

Css

       .capitalize{
            text-transform: capitalize;
      }
        .lowercase{
          text-transform: lowercase;
      }
      .uppercase{
          text-transform: uppercase;
      }
      .none{
          text-transform: none;
      }
      .inherit{
          text-transform: inherit;
      }

Html

          <p><span class="capitalize">This is an example. Example for capitalize.</span></p>
          <p><span class="lowercase">This is an example for lowercase.</span></p>
          <p><span class="uppercase">This is an example for uppercase.</span></p>
          <p><span class="none">This is an example for none.</span></p>
          <p><span class="inherit">This is an example for inherit.</span></p>

Результат:

This is an example. Example for capitalize.

This is an example for lowercase.

This is an example for uppercase.

This is an example for none.

This is an example for inherit.

Властивість text-decoration

Властивість text-decoration задає вигляд декоративних рядків на тексті.

  • text-decoration-color- встановлює колір декорацій, доданих до тексту за допомогою текстового оформлення-рядка.
  • text-decoration-line- встановлює тип декорування, який використовується на тексті в елементі, наприклад, підкреслення або надмірне виділ.
  • text-decoration-style- встановлює стиль рядків, визначених рядком текстового оформлення.
  • text-decoration-thickness- встановлює товщину обведення лінії декорування, яка використовується на тексті в елементі, наприклад, на лінії, підкреслення або надрядковому.

Синтаксис:

Css

 
        .text-decoration-color{
            text-decoration-color: lavender;
        }
        .text-decoration-line{
            text-decoration-line: line-through;
        }
        .text-decoration-style{
            text-decoration-style: dashed;
        }
        .text-decoration-thickness{
            text-decoration-thickness:inherit;
        }

Html

          <p><span class="text-decoration-color">This is an example.</span></p>
          <p><span class="text-decoration-line">This is an example.</span></p>
          <p><span class="text-decoration-style">This is an example.</span></p>
          <p><span class="text-decoration-thickness">This is an example.</span></p>

Результат:

This is an example.

This is an example.

This is an example.

This is an example.
